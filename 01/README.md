# `#01` PHP: JavaScript, The Basics


Do pliku HTML dodaj plik `app.js` i sprawdź w konsoli (Chrome -> F12), czy widzisz zawartość zmiennych.

Następnie wykonaj poniższe zadania:
- Sprawdź, jakiego typu są to zmienne. Użyj `typeof`.
- Wykonaj działania dla operatora dodawania i odejmowania: 
    - każdy wynik zapisz do osobnej zmiennej,
    - sprawdź, czy wyniki są prawidłowe:
        - jeśli tak, nic nie rób,
        - jeśli nie, rozwiąż problem (użyj np. [`parseInt()`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/parseInt), [`Number()`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Number) lub [`.toFixed()`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Number/toFixed)).
- Użyj instrukcji warunkowej, aby sprawdzić, który z wyników jest większy od 5, a który mniejszy.



> W przypadku języka programowania JavaScript nie trzeba wykorzystywać serwera do uruchamiania kodu. Środowiskiem uruchomieniowym jest przeglądarka - wystarczy uruchomić plik HTML, do którego jest podpięty plik `.js` (przy pomocy [tagu script](https://developer.mozilla.org/en-US/docs/Web/HTML/Reference/Elements/script)), aby kod źródłowy został zinterpretowany i uruchomiony.
